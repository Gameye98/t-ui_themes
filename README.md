## HOW TO INSTALL THEMES

With the great help of ĐɆ₳ĐⱤ₳฿฿ł₮ (@deadrabbit404) and Arun (@killadinjan), I wrote a script to automatically download and install all the themes available in this repository - via T-UI Expert.<br>
<br>
TTM script can be found in **[theme-manager folder](https://github.com/M4dGun/t-ui_themes/tree/main/theme-manager)** : please report any error via **[GitHub issues](https://github.com/M4dGun/t-ui_themes/issues)**  or via **[Telegram.](https://t.me/tuilauncher)** Thank you.<br>

❗️ **DOWNLOAD TTM SCRIPT INSTALLER** from **[here.](https://github.com/M4dGun/t-ui_themes/tree/main/theme-manager)**<br>
